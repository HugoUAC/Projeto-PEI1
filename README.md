# Projeto-PEI1
Grupo 10 - Hugo Raposo, Julia Freitas, Sara Freire
No projeto PEI1 é pedido para ser criado uma landing page de uma web-page para um centro
académico clínico fictício nos açores.

Na etapa de design:
  • Wireframes - É possível ver vários tipos de organização, mas acabamos por não usar
nenhum deles, visto que a landing page seria simples e não precisaria de menus tão
complexos. No entanto, mantivemos as localizações do logotipo e do nome da clínica,
bem como as informações de rodapé.

  • Mockups - Ficou semelhante com o website final, mas com falta das restantes secções,
tendo apenas a parte da Missão/Apresentação.
A landing page final ficou com tons de azul, uma imagem de fundo de uma paisagem açoriana
bem como uma organização simples. Porém, estas escolhas foram pensadas, foi usado o azul para
haver uma relação entre a clínica e a UAC e a sua organização para ser mais fácil de navegar. Nós
tentamos manter as caixas de texto e tipografia simples e fáceis de entender para não causar
confusão ao utilizador. Além disso, foi usado keyframes para animações com imagens e certas
caixas de texto, para o site ficar mais dinâmico. Finalmente, foi feito com que o cabeçalho e o
rodapé ficassem sempre visíveis para o utilizador, caso queira ir para diferentes secções do
website.
